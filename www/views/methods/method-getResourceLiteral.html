getResourceLiteral

Gets the text/message corresponding to the given key and locale


Returns: [String] Localized text.

Since: v2.0

Author: Hector Rosell


Parameters

Key [String]: to match text

locale [locale]: The locale to get localized message, or the locale descriptor ("language" or "language-country" two-letters ISO codes).


country [String]: A valid ISO Country Code.
language [String]: A valid ISO Language Code.
